Project Overview

Tổng quan hệ thống

Website tập trung vào bài toán regression: dự đoán giá đất ở Đà Nẵng theo đơn giá triệu/m², sau đó quy đổi ra tổng giá dựa trên diện tích lô đất.

Dữ liệu đầu vào

Dữ liệu chỉ lấy các feature liên quan đến đặc điểm của lô đất.

Đặc trưng sử dụng

Mô hình dùng diện tích, mặt tiền, độ rộng đường trước đất và mã quận/huyện để học quan hệ với đơn giá đất.

Mô hình chính

Linear Regression được xây dựng bằng NumPy và huấn luyện bằng Gradient Descent. Random Forest chỉ dùng để benchmark trong phần Analytics.

Linear Regression

Dự đoán giá đất bằng mô hình tuyến tính

Khi cần nhập thông tin lô đất và xem giá dự đoán, hãy mở trang Linear Regression. Trang đó chứa form nhập dữ liệu, kết quả dự đoán và các thông tin chính của mô hình.

Đi tới Linear Regression

Analytics

Đánh giá và so sánh mô hình

Trang Analytics trình bày MAE, RMSE, MSE, R², biểu đồ Loss, Actual vs Predicted và bảng so sánh Linear Regression với Random Forest.
